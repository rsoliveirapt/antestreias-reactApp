# Antestreias — Gestor e Portal de Cinema

![Antestreias Banner](banner.png)

O **Antestreias** é uma plataforma e portal de cinema concebida para a publicação de antestreias, passatempos, notícias do mundo cinematográfico, críticas e trailers.

---

## 🚀 Funcionalidades Principais

* **Frontend Moderno e Reativo:** Desenvolvido em React com TypeScript, empacotado pelo Vite, proporcionando carregamentos instantâneos e transições fluidas.
* **API de Alto Rendimento:** Backend estruturado em PHP puro ligando à base de dados MySQL via PDO.
* **Gestão de Passatempos:** Sistema de participação e registo de vencedores com formulários interativos.
* **Perfis e Críticas:** Secção de avaliações de filmes com controlo de moderação e perfis de utilizador.
